🎮 Hangman Game (Adam Asmaca) - C# Windows Forms

Bu proje, C# Windows Forms kullanılarak geliştirilmiş klasik Adam Asmaca (Hangman) oyunudur. Oyuncu, gizli kelimeyi harf harf veya kelime tahmini yaparak bulmaya çalışır.

🚀 Özellikler
🎯 Harf tahmini sistemi
🧠 Kelimeyi komple tahmin etme özelliği
❤️ Can sistemi (yanlış tahminlerde azalır)
😄 Emoji tabanlı adam asmaca görselleştirmesi
🏆 Skor sistemi
🔄 Yeni oyun başlatma özelliği
📊 Girilen harfleri takip etme
🎮 Oyun Mantığı
Doğru harf tahmini → kelime açılır
Yanlış tahmin → can azalır ve emoji değişir
Kelime tamamen bulunursa oyun kazanılır
Can bitince oyun kaybedilir
💡 Amaç

C# Windows Forms üzerinde temel oyun mantığını geliştirmek, döngüler, koşullar ve event yapısını öğrenmek.

🛠️ Kullanılan Teknolojiler
C#
Windows Forms (.NET Framework)
